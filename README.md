# DATA115
DATA115 Personal Data Set
The purpose of this project is to explore and analyze data sets that may contain information regarding how to ensure security and efficiency in communications systems. In the data set that I obtained, the CSRIC's best practices suggestions are organized by priority, with the most critical suggestions having a priority value of 3, and the least critical suggestions having a priority value of 1.
